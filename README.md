# Local_Food_Wastage_Management_System
Streamlit-based Food Redistribution System that analyzes food listings, claims, providers, and receivers to support efficient food donation and reduce food wastage.

# 🍽️ Food Redistribution Dashboard

## 📌 Project Overview

The **Food Redistribution Dashboard** is a data-driven platform designed to reduce food wastage by efficiently connecting food providers with receivers. The system collects and analyzes food donation data to provide meaningful insights and improve food distribution logistics.

This project demonstrates how **data analytics, SQL queries, and interactive dashboards** can help solve real-world problems such as food waste management.

The application allows users to explore datasets, analyze food donation patterns, and generate insights through SQL-based analytics.

---

# 🎯 Project Objectives

* Reduce food wastage by improving redistribution logistics
* Provide analytical insights into food donations and claims
* Enable data-driven decision making for food distribution networks
* Build an interactive analytics dashboard using Python and Streamlit

# 📊 Key Features

### 📦 Data Management

* Load and clean datasets for:

  * Food listings
  * Providers
  * Receivers
  * Claims

### 📈 Interactive Dashboard

* View and filter multiple datasets
* Explore data through interactive tables
* Dynamic filtering based on dataset attributes

### 📊 Analytics Using SQL

The dashboard provides insights such as:

* Total food available
* Providers per city
* Receivers per city
* Most common food type
* Provider with the most claims
* Average food provided by each provider
* City with highest food listings
* Food type claimed the most

### ⚠️ Expiry Alerts

The system automatically detects food items that are **expiring within the next 48 hours** and highlights them to ensure quick redistribution.

---

# 📂 Dataset Structure

The project uses four main datasets:

* **Food Listings** – Information about available food items
* **Providers** – Organizations donating food
* **Receivers** – NGOs or individuals claiming food
* **Claims** – Records of food claims by receivers
